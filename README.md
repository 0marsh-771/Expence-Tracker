# Expence Tracker
A simple and dynamic expense tracking web application built with vanilla JavaScript. 
Users can add income and expense transactions, track their balance, and manage their financial activity in real time.
## Features
Create Income and Expense transactions

-Automatic balance calculation

-Separate income and expense lists

-Data persistence using LocalStorage

-Create transaction using Enter key

-Input validation with error alerts

-Delete single transaction

-Clear all transactions

-Confirmation modal ("Are you sure?")
## Built With
-HTML

-CSS

-Vanilla JavaScript

-LocalStorage API
